# 개인정보처리방침

시행일: 2026-02-19

아파트 구구(이하 "앱")는 회원가입/로그인 없이 사용할 수 있는 계산 서비스입니다. 본 방침은 앱이 어떤 데이터를 처리하는지 명확히 안내합니다.

## 1. 수집하는 개인정보
- 앱은 이용자를 식별할 수 있는 개인정보(이름, 이메일, 전화번호, 주민번호 등)를 수집하지 않습니다.
- 회원가입/로그인 기능이 없으므로 계정 정보도 수집하지 않습니다.

## 2. 입력 데이터 처리 방식
- 이용자가 입력한 계산값(예: 연소득, 보유자산, 기존대출 월 상환액 등)은 계산 기능 제공을 위해 기기 내 저장소(localStorage)에만 저장될 수 있습니다.
- 해당 데이터는 개발자 서버로 전송되지 않습니다.

## 3. 분석/추적
- 앱은 광고 식별자(IDFA) 기반 추적을 수행하지 않습니다.
- 제3자 행동 추적 SDK를 사용하지 않습니다.

## 4. 제3자 제공 및 처리위탁
- 개인정보를 제3자에게 제공하지 않습니다.
- 개인정보 처리 위탁을 하지 않습니다.

## 5. 보관 및 삭제
- 로컬 저장 데이터는 사용자가 앱을 삭제하거나 기기 설정에서 앱 데이터를 삭제하면 함께 제거됩니다.

## 6. 아동의 개인정보
- 앱은 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

## 7. 이용자 권리 및 문의
- 사용자는 언제든지 앱 삭제 또는 데이터 삭제를 통해 로컬 데이터를 제거할 수 있습니다.
- 문의/요청: GitHub Issues
  - https://github.com/wani3000/Apartment-price-calculation/issues

## 8. 방침 변경
- 본 방침이 변경될 경우 본 문서에 시행일을 갱신하여 공개합니다.

## 9. 문서 원본 링크
